How to run application
Install Python and Flask: Make sure you have Python installed. Then, install Flask:
pip install Flask Werkzeug

Initialize the Database: Run the database.py script once to create the database.db file and populate it with initial data.
python database.py

Run the Flask App: Execute the app.py script.
python app.py

Access the Website: Open your web browser and go to http://127.0.0.1:5000/.
